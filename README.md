# 🌾 FarmX - Digital Agriculture Platform

**Connecting Farmers, Markets, and Technology for Sustainable Agriculture.**

FarmX is a comprehensive digital ecosystem designed to empower farmers, agribusinesses, and consumers in Nigeria and beyond. It provides a seamless marketplace, financial tools, job opportunities, and smart farming integrations all in one place.

🔗 **Live Demo:** [farmx-app.vercel.app](https://farmx-app.vercel.app) |

---

## 🚀 Key Features

### 🛒 Marketplace
- Buy and sell agricultural products (Crops, Livestock, Equipment, Seeds).
- Post ads with image uploads, pricing, and delivery options.
- Multiple payment methods (Wallet, Debit Card, Bank Transfer).

### 💰 Wallet & Finance
- Secure digital wallet for farmers and companies.
- Deposit, withdraw, and transfer funds seamlessly.
- Transaction history and spending analytics.
- Agri-Credits and Loan applications.

###  Jobs Board
- Browse and post agricultural jobs (Full-time, Part-time, Contract).
- Direct application system with employer notifications.

###  Knowledge Hub (Learn)
- Video tutorials and courses on modern farming techniques.
- Certifications and learning streaks to encourage continuous education.

### 🏢 Company Profiles
- Custom profile pages for agribusinesses (`www.companyname.farmx.com`).
- Showcase products, investments, followers, and events.
- Direct ordering system synced with the main app.

### 🤝 Community & Chat
- Social feed for farmers to share updates and connect.
- AI-powered chat assistant for instant farming advice.
- Real-time messaging system.

### 📊 Enterprise Dashboard
- **Analytics:** Track sales, views, and revenue.
- **Fleet Management:** Monitor trucks, tractors, and drones.
- **Inventory:** Track stock levels across warehouses.
- **Staff Management:** Manage employee roles and contacts.
- **Partners Hub:** Connect with NGOs, Government, and Corporate partners.

### ️ Smart Farming Integrations (Planned)
- Real-time weather and climate data.
- IoT sensor integration for soil and crop monitoring.
- Blockchain traceability for supply chain transparency.

## 🛠️ Tech Stack

### Current (Frontend Web)
- **HTML5** - Semantic structure
- **Tailwind CSS** - Utility-first styling (Custom FarmX theme: Red `#b71c1c`, White, Black)
- **Vanilla JavaScript** - Interactivity, LocalStorage for state management
- **Material Symbols Rounded** - Clean, modern iconography
- **Inter Font** - Balanced typography (400, 500, 600 weights)

### Planned (Backend & Mobile)
- **Supabase / AWS RDS** - Database and Authentication
- **Node.js / Express** - RESTful API
- **React Native (Expo)** - iOS and Android mobile apps
- **AWS S3 & Lambda** - File storage and serverless functions

## 🏃 Getting Started

### Prerequisites
- A modern web browser (Chrome, Safari, Firefox).
- No build tools required for the current web version!

### Running Locally
1. Clone the repository:
```bash
   git clone https://github.com/smbabanbaba-maker/farmx-app.git
```
2. Navigate to the project folder:
```bash
   cd farmx-app
```
3. Open `index.html` in your browser, or use a local server like Live Server in VS Code.
